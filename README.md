# pierwszy_projekt

A simple Python project that can be run directly in PyCharm.

## Description

This project contains a basic Python script (`main.py`) that demonstrates:
- User input handling
- Simple greeting functionality
- Basic calculations
- Error handling

## Requirements

- Python 3.6 or higher
- PyCharm (Community or Professional Edition)

## How to Run in PyCharm

### Method 1: Run directly
1. Open this project in PyCharm
2. Open the `main.py` file
3. Right-click anywhere in the editor
4. Select "Run 'main'" from the context menu
5. The script will execute in the PyCharm Run console

### Method 2: Using the Run button
1. Open this project in PyCharm
2. Open the `main.py` file
3. Click the green "Run" arrow (▶) in the top right corner
4. Select "Run 'main'" from the dropdown menu

### Method 3: Using keyboard shortcut
1. Open this project in PyCharm
2. Open the `main.py` file
3. Press `Shift + F10` (Windows/Linux) or `Ctrl + R` (macOS)

## What the Script Does

When you run the script, it will:
1. Display a welcome message
2. Ask for your name and greet you
3. Ask for two numbers and calculate their sum
4. Display the result and a goodbye message
